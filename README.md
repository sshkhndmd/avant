# AVANT — интернет-магазин одежды с интеграцией VK Mini App

**AVANT** — дипломный проект интернет-магазина для бренда одежды.
Проект включает полноценный веб-сайт и отдельную версию в формате **VK Mini App**, которые работают с одним backend-сервером и общей базой данных.

## О проекте

Цель проекта — разработать современный интернет-магазин с каталогом товаров, корзиной, избранным, оформлением заказов и административной частью для управления товарами и пользователями.

Проект реализован как клиент-серверное приложение:

* основной сайт на **React**;
* VK Mini App для пользователей внутри ВКонтакте;
* backend на **Node.js + Express**;
* база данных **PostgreSQL**;
* взаимодействие frontend и backend через **REST API**.

## Основной функционал

### Пользовательская часть

* просмотр каталога товаров;
* фильтрация товаров по категориям;
* просмотр карточки товара;
* выбор размера;
* добавление товара в корзину;
* добавление товара в избранное;
* оформление заказа;
* регистрация и авторизация;
* просмотр истории заказов;
* отмена заказа.

### Административная часть

* управление товарами;
* управление пользователями;
* разграничение доступа по ролям;
* просмотр и обработка заказов;
* работа с возвратами.

### VK Mini App

В проекте также реализована отдельная версия интерфейса для VK Mini App.
Она использует тот же backend и ту же базу данных, что и основной сайт.

В VK Mini App доступны основные пользовательские функции:

* главная страница;
* каталог;
* карточка товара;
* корзина;
* избранное;
* профиль;
* история заказов.

## Архитектура проекта

Проект состоит из двух клиентских приложений и одного backend-сервера.

```text
AVANT
├── frontend          # React-приложение
├── avant-backend     # Backend на Node.js + Express
└── database          # SQL-структура базы данных
```

Обе клиентские части обращаются к одному REST API:

```text
React Web App  ─┐
                ├── REST API ── PostgreSQL
VK Mini App   ──┘
```

## База данных

В проекте используется PostgreSQL.
Основные таблицы:

* `users` — пользователи;
* `products` — товары;
* `product_images` — изображения товаров;
* `product_sizes` — размеры и остатки;
* `favorites` — избранные товары;
* `cart` — корзина;
* `orders` — заказы;
* `order_items` — товары в заказе;
* `returns` — возвраты.

## Безопасность и авторизация

Для авторизации используется JWT-токен.
После входа пользователь получает токен, который хранится на клиенте и передаётся в запросах к защищённым маршрутам.

Пароли пользователей не хранятся в открытом виде.
Для хеширования используется `bcrypt`.

Также реализовано разграничение доступа по ролям:

* пользователь;
* администратор;
* менеджер товаров;
* менеджер цен.

## Оформление заказов

При оформлении заказа backend проверяет наличие товара и выбранного размера.
Для защиты от ситуации, когда два пользователя одновременно покупают последний товар, используется транзакция в PostgreSQL.

Это позволяет корректно резервировать остатки и не допускать продажи товара, которого уже нет в наличии.

## Установка и запуск проекта

### 1. Клонирование репозитория

```bash
git clone https://github.com/your-username/avant.git
cd avant
```

### 2. Установка backend-зависимостей

```bash
cd avant-backend
npm install
```

### 3. Создание `.env` файла

В папке backend необходимо создать файл `.env`:

```env
PORT=3000
DATABASE_URL=postgresql://username:password@localhost:5432/avant_store
JWT_SECRET=your_jwt_secret
```

### 4. Запуск backend

```bash
npm start
```

или в режиме разработки:

```bash
npm run dev
```

### 5. Установка frontend-зависимостей

```bash
cd ../frontend
npm install
```

### 6. Запуск frontend

```bash
npm run dev
```

После запуска frontend будет доступен по адресу:

```text
http://localhost:5173
```

Backend будет доступен по адресу:

```text
http://localhost:3000
```
